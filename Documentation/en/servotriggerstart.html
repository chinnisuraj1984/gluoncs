SERVO TRIGGER START

This command will start continuous triggering of the servo (e.g. to trigger a camera). It can be stopped by using SERVO_TRIGGER_STOP.

This sequence allows you to, for example, use a servo to push a camera. button.

Parameters:

  • Servo channel: 1 to 8. Channels 5 and 6 are usually used. 7 and 8 are available on the pins behind number 5 and 6 (only the signal part)

  • Pulse width: a servo is controlled by sending a pulse between 1000 and 2000µs. This value determines the servo's position.
    A value of "0" will output a logical 0 (GND level) on this channel's output. A value of "2500" or higher will output a logical 1 (5V).

  • Time between triggers: the time in milliseconds between every servo trigger.

  • Trigger mode: (1) PWM if you're using a real servo to trigger, or trigger which accepts PWM servo pulses as input. (2) CHDK mode: connect the servo output to the CHDK USB input. This will put the output constantly on "high", so you will need a CHDK script to start continuous triggering when the input is "high". When in CHDK mode, only the servo channel parameter is being used.
